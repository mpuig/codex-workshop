# Data Catalog

This page documents every data file used in the three exercises. Use it as a quick reference when working with the data or when designing prompts for Codex.

All data is **synthetic** -- generated for training purposes. No real customer, policyholder, or claims information is included.

**Format:** JSON | **Sections:** 7

| Section | Description |
|---------|-------------|
| `category_mapping` | Rules mapping product_line + incident_type to claim categories |
| `priority_rules` | Criteria for assigning High, Medium, or Low priority |

| `team_assignment` | Rules for routing claims to the correct handling team |

| `fraud_indicators` | Red flags and patterns that trigger fraud investigation |
| `region_codes` | Reference table of region codes and their full names |

### report_template.md

**Format:** Markdown template

A structured report template with placeholder sections for:

1. Executive Summary
2. Key Performance Indicators (with KPI definitions table)
3. Claims analysis breakdowns (by product line, region, status)
4. Fraud monitoring
5. Trends and recommendations

The template includes `[PLACEHOLDER]` markers where Codex should insert computed values and narrative.
